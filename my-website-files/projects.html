PROJECTS

Cloud-Native Portfolio (This Website)

Architected and deployed a serverless portfolio website on AWS to demonstrate full-stack cloud engineering skills. The frontend is hosted on S3 and delivered globally via a secure CloudFront CDN with a custom domain. Currently developing the backend API using Python (Lambda), API Gateway, and DynamoDB to power a real-time visitor counter. The entire infrastructure is being defined as code using Terraform and automated via a GitHub Actions CI/CD pipeline.

Automated CI/CD Pipelines for AWS Container Services

Designed and implemented two fully automated CI/CD pipelines to deploy a containerized Python web application onto distinct AWS architectures. The first pipeline deployed to a production-grade ECS Fargate environment behind an Application Load Balancer, with all networking and service infrastructure managed via Terraform. As a cost-effective alternative, engineered a second, fully serverless pipeline using AWS App Runner, automating deployments and SSL management. Both solutions utilized GitHub Actions for CI/CD and OIDC for secure, keyless authentication to AWS.

Personal Homelab

Architected and deployed a private, end-to-end encrypted, real-time sync service on a headless Arch Linux server to provide a secure, self-hosted cloud solution for personal data. The service operates by running a containerized CouchDB database managed by Docker and Docker-Compose. The deployment involved securing the service at the network layer through ip-tables and Docker firewall.
